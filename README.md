## 项目实践

### 沃土云脑 - 多模态农田智能检测与决策系统

**省级大学生创新创业训练计划项目** ｜ 2025.03 - 2025.06

围绕田区环境下的农作物病虫害识别问题，负责图像分类模型的设计与实现。构建农作物病虫害图像数据集，并利用 WGAN 进行样本扩充，以缓解样本数量不足与类别不均衡问题。随后采用 MobileNetV2、MobileNetV3 等轻量级卷积神经网络作为分类主干，通过迁移学习与冻结层微调策略提升小样本条件下的泛化能力。

**成果：**

- 计算机设计大赛西北赛区三等奖
- 两项软件著作权

### 千亿赛道与指尖经济 - 网络微短剧市场现状调查与消费行为影响因素分析

**数据分析与市场研究实践项目** ｜ 2024.12 - 2025.04

围绕网络微短剧市场现状及消费者行为影响因素展开研究，主要负责问卷数据整理、统计分析与结果解读工作。基于 Python、SPSS、AMOS 等工具完成问卷数据预处理、描述性统计分析、信度与效度检验、相关性分析、回归分析及结构方程模型分析，挖掘用户消费意愿、观看偏好与行为特征之间的关系。

**成果：**

- 第十五届全国大学生市场调查与分析大赛国家级三等奖
